# currencyConverter  
Requirements:  
Java            17+  
Maven           3.5+  

Description:   
Just a currency converter calculator  
    
How to run:  
1 - install jdk 17+  
2 - install maven 3.5+  
3 - git clone https://github.com/hey-world-team/currencyConverter.git  
4 - mvn spring-boot:run  
5 - go to http://localhost:8080/
